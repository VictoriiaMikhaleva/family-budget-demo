# family-budget-demo

Демо-версия калькулятора семейного бюджета. Данные хранятся только в **localStorage** браузера (ключ `family-budget-demo-v1`), без синхронизации с Firebase.

Участники — Гайка, Рокфор, Чип, Дейл и Вжик. Аватары — стилизованные SVG (не официальные кадры Disney).

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала (обычно `http://localhost:5173`).

Сборка: `npm run build`, предпросмотр: `npm run preview`.

## Возможности

- Учёт доходов и расходов по участникам и категориям
- Графики, лимиты, история, архив месяцев, подсказки
- Экспорт CSV/JSON и импорт резервной копии
