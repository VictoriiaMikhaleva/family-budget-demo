# family-budget-demo

Демо-версия калькулятора семейного бюджета. Данные хранятся только в **localStorage** браузера (ключ `family-budget-demo-v2`), без синхронизации с Firebase.

Участники — Сима, Рыжик, Умка, Плюша и Снежок. Аватары в `public/avatars/`.

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала (обычно `http://localhost:5173`).

Сборка: `npm run build`, предпросмотр: `npm run preview`.

## Возможности

- Учёт доходов и расходов по участникам и категориям
- Графики, лимиты, история, архив месяцев, подсказки
- Экспорт CSV/JSON и импорт резервной копии
